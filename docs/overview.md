# 제품 개요

## 한 줄

**몽글(Mongle)** — 영화와 감정을 부드럽게 기록하는 시네마틱 공간.

## 브랜드

- **한글:** 몽글  
- **영문:** Mongle (서브 라벨로만 사용)  
- **톤:** 포근함, 잔잔한 여운, 진주빛·클레이 질감의 소프트 프리미엄 UI  
- **심볼:** 3D 진주빛 클라우드 (`public/cloud.png`) — 파비콘·앱바에 사용

UI 카피는 **한글을 기본**, 섹션 영어 라벨(`Continue Watching` 등)은 보조로만 둡니다.

## 무엇을 해결하나

일반적인 OTT 탐색은 “무엇이 핫한가”에 집중합니다.  
몽글은 **지금 느낌에 맞는 작품**을 고르고, 본 뒤의 **감정을 한 줄 저널로 남기는** 경험에 초점을 둡니다.

| 영역 | 의도 |
|------|------|
| 홈 | 오늘의 추천과 이어보기로 부드럽게 시작 |
| 탐색 | 감정·분위기 칩으로 작품을 고름 |
| 상세 | 시네마틱 백드롭 위에서 줄거리·출연 확인 |
| 아카이브 | 감정 버블이 달린 저널 카드로 기록 회고 |
| 프로필 | 나의 몽글 활동 요약 |

## 현재 스코프 (토이)

- 작품 목록은 **mock** (`src/data/mock.ts`)
- 포스터/백드롭은 TMDB **이미지 CDN**
- 저널은 **Supabase** + **Google 로그인** (기기 간 동기화)

## 핵심 사용자 여정

1. 홈에서 오늘의 추천을 보고  
2. 감정으로 탐색하거나 작품을 고르고  
3. Google로 로그인한 뒤 “몽글 기록하기”  
4. 아카이브·프로필에서 같은 기록을 확인
