# 제품 개요

## 한 줄

**몽글(Mongle)** — 영화와 감정을 부드럽게 기록하는 시네마틱 공간.

**Live:** [mongle-steel.vercel.app](https://mongle-steel.vercel.app)

## 브랜드

- **한글:** 몽글  
- **영문:** Mongle (서브 라벨로만 사용)  
- **톤:** 포근함, 잔잔한 여운, 진주빛·클레이 질감의 소프트 프리미엄 UI  
- **심볼:** 3D 진주빛 클라우드 (`public/cloud.png`) — 파비콘·앱바에 사용

UI 카피는 **한글을 기본**, 섹션 영어 라벨(`Trending Now` 등)은 보조로만 둡니다.

## 무엇을 해결하나

일반적인 OTT 탐색은 “무엇이 핫한가”에 집중합니다.  
몽글은 **지금 느낌에 맞는 작품**을 고르고, 본 뒤의 **감정을 한 줄 저널로 남기는** 경험에 초점을 둡니다.

| 영역 | 의도 |
|------|------|
| 홈 | TMDB 트렌딩·분위기 레일로 부드럽게 시작 |
| 탐색 | 검색·감정 칩으로 작품을 고름 |
| 상세 | 시네마틱 백드롭, 위시리스트, 기록하기 |
| 아카이브 | 공개 피드, 감정 버블 카드, 좋아요·수정·삭제 |
| 위시리스트 | 나중에 볼 작품 모음 |
| 프로필 | 닉네임·이모지, 몽글 통계 |

## 현재 스코프 (토이)

- 작품 데이터: **TMDB API** (홈·탐색·상세), mock은 폴백·시드용  
- 저널·위시리스트·좋아요: **Supabase** + **Google 로그인**  
- 배포: **Vercel** (`mongle-steel.vercel.app`)

## 핵심 사용자 여정

1. 홈·탐색에서 작품을 고르고  
2. Google로 로그인한 뒤 “몽글 기록하기”  
3. 아카이브에서 내·다른 사람 기록을 보고 좋아요를 남기고  
4. 위시리스트·프로필에서 활동을 이어감
